# Diamond Sutra Translation Stylometry

An anonymized replication package for a stylometric study of Chinese Buddhist translations.

## Repository Contents

- The_Diamond_Sutra.ipynb
- kumarajiva.docx
- xuanzang.docx
- PCA coordinates
- PCA loading tables
- Entropy statistics
- Generated figures

## Data Source

The source texts were obtained from CBETA.

## Reproducibility

Run the notebook in Google Colab or Jupyter Notebook.

The notebook reproduces:

- Character-frequency extraction
- Sliding-window segmentation
- PCA analysis
- Loading interpretation
- Shannon entropy analysis
- Statistical testing
